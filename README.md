# TopTwenty
top twenty films ionic app

<h1>Start app</h1>
$ git clone https://github.com/DSergii/TopTwenty.git && cd TopTwentyIonicApp <br />
``` javascript 
- npm install
- bower install
if not install ionic use 
- npm install -g cordova ionic
```
- for testing: ionic serve --lab
- for update: ionic update
