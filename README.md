# TopTwenty
top twenty films ionic app

<h1>Start app</h1>
$ git clone https://github.com/DSergii/TopTwenty.git && cd TopTwentyIonicApp <br />
``` javascript 
$ npm install
$ bower install
```
<p>if not install ionic - use</p>
```javascript
$ npm install -g cordova ionic
```
<p>for testing:</p>
```javascript
$ ionic serve --lab
```
<p>for update:</p>
```javascript
$ ionic update
```
